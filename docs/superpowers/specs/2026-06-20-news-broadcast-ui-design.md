**版本**：v1.0

## 1. 设计决策汇总

| # | 决策项 | 结论 |
|---|--------|------|
| 1 | 视觉风格 | 4套主题可切换，默认「温暖陪伴」（豆包式亲和感） |
| 2 | 导航结构 | 四Tab底部导航（新闻/播客/追问/我的）+ 可拖拽浮动FAB |
| 3 | 卡片交互 | 纯横向滑动 Tinder式，左右滑动切换新闻 |
| 4 | 播客播放器 | 全屏沉浸式，音乐播放器歌词模式（居中大字幕+播放控制） |
| 5 | 追问面板 | 移动端：底部半屏抽屉；桌面端：可收起侧栏 |
| 6 | 桌面端布局 | 侧栏导航 + 多栏展开（新闻双列卡片 / 播客歌词+章节双栏） |
| 7 | 动效级别 | 中等丰富（关键交互动效 + 过渡动画 + 骨架屏 + hover微交互） |
| 8 | FAB按钮 | 可拖拽、松手自动贴边吸附、默认半透明 |

---

### 1.1 v1.1 交互修订（2026-06-23）

| 模块 | 修订要求 |
|------|----------|
| 新闻详情 | 点开卡片后不能复用卡片摘要；详情页必须展示深度解读、关键事实、背景、影响、来源链接和可信度提示 |
| 追问 FAB | 默认贴边半隐藏，只露出可点击区域；支持自由拖动，拖动过程跟手，松手自动吸附左右边；拖动和点击必须可靠区分 |
| 追问抽屉 | 除手动文字输入外，需要有“连续聆听”模式，状态清楚显示：待机/聆听中/识别中/回答中/播放回答 |
| 播客消费 | 播客页打开时应直接展示最近一期已生成内容，不出现长时间“生成中”阻塞 |
| TTS 声音 | 正式体验不使用机械感明显的兜底音色；播客双主播和追问回答使用不同语气/音色 |

## 3. 信息架构

```
NewsCast
├── 📰 新闻卡片页 (CardNewsPage)
│   ├── 卡片堆叠容器 (CardStack)
│   ├── 单张新闻卡片 (NewsCard) ×N
│   │   ├── 分类标签
│   │   ├── 新闻配图（可选）
│   │   ├── 标题 + 摘要
│   │   └── 来源 + 时间
│   └── 进度指示器
│
├── 🎧 播客页 (PodcastPage)
│   ├── 全屏播放器 (PodcastPlayer)
│   │   ├── 歌词式字幕滚动区（区分主播A/B颜色）
│   │   ├── 播放控制（播放/暂停/快进/后退/倍速）
│   │   └── 音频进度条（可拖拽）
│   ├── 章节标记列表
│   └── 期数元信息
│
├── 💬 追问页 (AskPage) / 追问抽屉 (AskDrawer)
│   ├── 对话气泡列表（用户/AI区分左右）
│   ├── 上下文标签（关联的新闻/播客话题）
│   ├── 输入区（文字输入 + 语音按钮）
│   └── AI 打字动画
│
└── 👤 我的页 (MePage)
    ├── 陪伴助手人格设置（名称/性格/音色/语速）
    ├── 主题切换（4套）
    ├── 兴趣偏好标签
    ├── 推送时间设置
    └── 关于/版本/邀请码
```

## 4. 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|

## 5. 交互规范（尼尔森十大原则映射）

| 原则 | 实现 |
|------|------|
| 1. 系统状态可见 | 进度指示器（第N/总共M条）、播放进度条、AI打字动画、骨架屏加载态 |
| 2. 系统与现实的匹配 | 自然语言对话、语音交互、新闻分类使用日常语言 |
| 3. 用户可控与撤销 | 滑动可回退、追问面板可关闭、播放可暂停/恢复 |
| 4. 一致性与标准 | 全局统一Tab结构、统一的卡片样式、统一的色彩语义 |
| 5. 防错 | 删除/清理操作二次确认、输入框为空时发送按钮禁用 |
| 6. 识别而非回忆 | 图标+文字标签双重识别、追问上下文标签显示当前话题 |
| 7. 灵活高效 | 快捷键预留、双击卡片进入详情、左右滑动手势（移动端）+ 键盘←→（桌面端） |
| 8. 简洁美观 | 卡片少而精、播客无干扰全屏、追问面板半屏不遮挡主内容 |
| 9. 帮助用户识别与恢复 | 网络错误Toast提示+重试按钮、空状态引导文案 |
| 10. 帮助文档 | 「我的」页内置简短使用引导 |

---

### 5.1 FAB 贴边隐藏规范

- 默认位置：移动端右侧中下区域，露出约 28px 宽度，避免遮挡新闻正文和底部 Tab。
- 拖动：使用 pointer/touch 直接跟随手指，不在拖动中使用过渡动画。
- 吸附：松手后根据按钮中心点自动吸附到左边或右边，保持半隐藏。
- 点击：移动距离小于 6px 才视为点击；超过阈值只移动，不打开抽屉。
- 可达性：半隐藏状态仍保留足够触控区域；按钮需要有无障碍标签。

## 6. 动效清单（inspira-ui）

| 位置 | 动效 | 触发 |
|------|------|------|
| 卡片滑动 | 卡片飞出（方向跟随手指）+ 下一张卡片从底部升入 | 滑动 |
| 追问抽屉 | 底部弹簧弹出（spring animation） | 点击FAB |
| Tab切换 | 内容区淡入淡出（crossfade 200ms） | 切换Tab |
| 播客字幕 | 当前行放大+高亮平滑过渡 | 播放进度 |
| 骨架屏 | 呼吸闪烁动画 | 数据加载中 |
| 按钮hover | 微放大 1.05x + 阴影增强 | hover |
| 主题切换 | 背景色/文字色 transition 400ms | 切换主题 |
